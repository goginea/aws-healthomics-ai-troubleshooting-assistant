# Installation Guide

## Prerequisites

- Kiro IDE installed
- AWS Account with HealthOmics access
- Node.js >= 18.0.0

## Installation Steps

1. Open Kiro IDE
2. Navigate to Powers marketplace
3. Search for "HealthOmics AI Troubleshooter"
4. Click Install
5. Install required dependency Powers when prompted
6. Follow the Setup Wizard
