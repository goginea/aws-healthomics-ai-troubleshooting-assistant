# Examples

This directory contains example workflows and failure scenarios for testing the HealthOmics AI Troubleshooter.

## Contents

- `workflows/` - Example Nextflow, WDL, and CWL workflow definitions
- `failures/` - Sample failure scenarios with expected root causes
- `policies/` - Sample IAM policy templates

## Usage

Use these examples to test the troubleshooting system and understand common failure patterns.
